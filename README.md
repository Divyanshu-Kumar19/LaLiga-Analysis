# LaLiga Teams Analysis

Comprehensive analysis of La Liga football teams spanning seven seasons (2019-2026). This project combines performance statistics, financial data, and squad information to provide insights into team dynamics and investment potential.

## Project Overview

This analysis examines La Liga teams across multiple dimensions:
- **Performance Metrics**: Match results, points per game, win rates, goal statistics, and expected goals (xG) analysis
- **Financial Analysis**: Stadium attendance, revenue patterns, and overall financial health scores
- **Squad Analysis**: Player market values, age distribution, squad depth, and quality metrics
- **Head-to-Head Comparisons**: Direct matchup statistics and historical performance between teams
- **Machine Learning Predictions**: ML-based investment scoring model for team performance forecasting
